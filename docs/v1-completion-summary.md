# V1 Completion Summary

This repository currently contains a local Django V1 foundation for the Brisbane Star Care NDIS admin system.

## Included

- Role-based login, redirects, and page access controls.
- Admin dashboard and worker dashboard.
- Participant records.
- Support worker records.
- Participant-worker assignments.
- Support item records.
- One-off and recurring shift scheduling.
- Worker shift views.
- Worker service log submission.
- Admin service log review.
- Invoice generation from approved logs.
- CSV and simple PDF invoice exports.
- Document upload, download, and worker document access.
- Audit logging for key business actions.
- Local environment settings and deployment checklist.

## Not yet included

- Production hosting.
- Final visual design or purchased template integration.
- Calendar-style roster UI.
- Notification delivery.
- Advanced reporting dashboards.
- Payroll integrations.
- External NDIS API integrations.

## Recommended next use

Use `docs/v1-trial-release-notes.md` and `docs/v1-qa-checklist.md` for local browser testing. After the checklist is stable, the project can move into UI refinement, reporting, or deployment preparation depending on business priority.
