# Phase 39 Roster Worker Filter Select Design

## Goal

Make the roster worker filter easier to use by replacing the worker ID text field with a worker name dropdown.

## Scope

- Show active support workers in the roster filter form.
- Preserve the existing `worker` query parameter and filtering behavior.
- Keep the selected worker selected after searching.
- Keep roster filter summary behavior unchanged.

## Out Of Scope

- Changing shift creation forms.
- Adding worker search autocomplete.
- Changing worker assignment or scheduling rules.
