# Phase 13 V1 QA Polish Design

Phase 13 is a local V1 readiness pass. It does not add a new business module; it makes the current system easier to review and prepares a clear checklist for browser testing.

## Goals

- Replace outdated dashboard placeholder copy with current V1 module summaries.
- Add dashboard entry links to existing admin and worker workflows.
- Document what V1 currently includes and what remains out of scope.
- Provide a practical QA checklist for local testing before deeper UI refinement or deployment preparation.

## Non-goals

- No final visual redesign.
- No purchased template integration.
- No production deployment.
- No new business workflow models.

## Acceptance criteria

- Admin dashboard references current modules instead of future phases.
- Worker dashboard points workers to shifts, logs, documents, and profile.
- V1 QA and completion summary documents exist.
- README links to the new V1 review documents.
- Automated tests verify dashboard copy does not regress to early placeholder text.
